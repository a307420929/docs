## 项目运行

**注意 ：由于涉及大量的 ES6/7 等新属性，node 需要 6.0 以上版本**

项目是用vue与webpack搭建的，前端vue打开项目是需要一个环境运行的。

```Terminal

git clone git@10.4.4.90:CyberSOC/swad-webUi.git

cd swad-webUi

npm install  或 yarn(推荐)

npm run dev
```
